Project 1: Australian Stock Exchange Real-Time Analytics Dashboard
Domain: Data Engineering + Cloud Architecture
Timeline: 5-7 days
Business Value: Financial services sector focus (highest recruiting in Australia)
Technical Stack:

Backend: Python with FastAPI, WebSocket connections
Data Sources: ASX API for real-time stock data, RBA economic indicators
Processing: Apache Kafka for streaming, Redis for caching
Frontend: React with real-time charts (Recharts/D3.js)
Deployment: AWS Lambda + API Gateway + CloudFront
Infrastructure: Terraform for IaC

Key Features:

Real-time ASX stock price monitoring for ASX 200 companies
Economic indicator correlation (RBA cash rate, inflation data)
Multi-region deployment (Sydney ap-southeast-2, Melbourne backup)
Cost optimization with serverless auto-scaling
Security: API key management, CORS, rate limiting

Australian Context Advantage: Uses local financial data, demonstrates understanding of Australian market regulations, shows cost-consciousness important to local businesses.
